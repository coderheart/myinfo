Codeigniter Multistep Form Wizard

Codeigniter Multistep Form Wizard is a Multistep PHP (Codeigniter Framework) based form wizard. It is fully responsive (Bootstrap framework) based and Ajax supported. It can be easily customized as it is well documented and developed using latest web technologies. This form will be a perfect choice for all registration and Loan Industry based websites.

Installation

Note: Note: Please check your PHP version before installing. In order to work properly, your PHP version should be at least 5.5. We recommend PHP 5.6 or above

  1. Go to Cpanel and Click MySQL® Databases

  2. Create New Database

  3. Add New User

  4. Add User To Database

  5. Set User Permissions

  6. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside public_html folder.
  7. Connect With Database:
    • Import database.sql file which given project folder
  8. Configure database connection:
    • Goto the extract folder and Find the file: application/config/database.php

      and open it using a editor like Notepad or others. and set your created database, username, password below

      •               'hostname' => 'your_host_name', 
              'username' => 'your_username', 
              'password' => 'your_password', 
              'database' => 'your_database_name'
  9. Run the Install file. (http://www.yourdomain.com/)

Customization

  1. Adding more step

    if You want add more step just copy tab section (class="tab")

    <div class="tab"> Your Code here.... </div>

    And also add this code (given below)

    <li class="indicator "><span></span><p>Your Text</p></li> <span class="step"></span>
  2. Validation field

    Just write required into input tag fields which your want to validate

    <input type="" name="" class="form-control" required />
  3. Customize in PHP file

    path: application/controllers/Home.php

    'database field name' => $this->input->post('html field name'),

    * Just add field in add_info function.

  4. No Need to customize Jquery file (if it not necessary).
